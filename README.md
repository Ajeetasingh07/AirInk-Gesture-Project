# AirInk Gesture Project ✍️

An AI-based air writing and gesture-controlled game using OpenCV and MediaPipe.

## 🚀 Features
- Air writing using hand gestures
- OCR letter recognition
- Gesture-based drawing
- Simple gesture game

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## ▶️ How to Run

```bash
pip install -r requirements.txt
python augmented.py
